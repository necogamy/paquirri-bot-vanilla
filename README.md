# **Paquirri Bot - helper desktop app**

## **Overview**
![screenshot](./resources/screenshot.png)
Paquirri helps you with daily tasks, for example:
* Credit Card Checker
* PC cache cleaner
* A simple GET method from an API

## **Built with**
* Vanilla Technologies:

	    + Node.js
	    + ECMAScript6 (JavaScript)
	    + CSS3
	    + HTML5
